app-estaciona

Neste projecto desenvolvemos uma aplicação mobile com a tecnologia Cordova.

Recuros utilizados
==============
1. HTML
2. Cordova
3. JavaScript
4. JQuery Mobile
5. JQuery
6. CSS

Instruções
============
1. Instalar as dependências do node
    ```bash
    npm install
    ```
1. Executar o aplicativo

    ```bash
    # No navegador
    cordova run browser
    
    # No android
    cordova run android
    ```
